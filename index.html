Execução

Assista o vídeo feito para o projeto Hello Java World em http://youtu.be/wa2zjLtc13A

Compilando os fontes

rm -fR bin

rm -fR dist

mkdir bin

mkdir bin/classes

shopt -s globstar

javac -cp libs/commons-cli-1.2.jar -d bin/classes **/*.java

Empacotando a aplicação

jar -cfm bin/app.jar ./src/resources/manifest.txt -C bin/classes .

jar -tf bin/app.jar
    META-INF/
        MANIFEST.MF
    how/
        Greetings.class
        App.class
        service/
            GreetingsService.class

mkdir dist

mkdir dist/bin

cp -R bin/app.jar dist/bin 

cp -R libs dist

Executando a aplicação

cd dist

java -jar bin/app.jar [options] <visitor-name>

Ex.: 

cd dist

java -jar bin/app.jar -fv Fulano
