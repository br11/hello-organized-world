Requisitos

  • Git
  • JDK 1.7.x

Baixando o código fonte

mkdir hands-on

cd hands-on

git clone https://github.com/br11enterprise/hello-organized-world.git

cd hello-organized-world

ls

Execução

Compilando os fontes

javac Greetings.java 

javac GreetingsService.java 

javac App.java

Compilando os fontes

rm -fR bin

rm -fR dist

mkdir bin

mkdir bin/classes

javac -cp libs/commons-cli-1.2.jar -d bin/classes **/*.java

Empacotando a aplicação

jar -cfm bin/app.jar ./src/resources/manifest.txt -C bin/classes .

mkdir dist

mkdir dist/bin

cp -R bin/app.jar dist/bin 

cp -R libs dist 

cp src/resources/run dist

Executando a aplicação

cd dist

java -jar bin/app.jar [options] <visitor-name>

Ex.: 

cd dist

java -jar bin/app.jar -fv Fulano
